\#### Pour Linux





Installer ADB ET FASTBOO ?????????????



sudo apt install adb fastboot mais cette commande ne donne pas la derniere version d'ADB et de Fastboot.



Testez avec avec \`adb version\` et \`fastboot --version\`





Et vérifiez la version d'ADB et de Fastboot installée avec \`adb version\` et \`fastboot --version\`.



La derniere version se trouge ici https://developer.android.com/tools/releases/platform-tools?hl=fr



Cliquer sur Télécharger la version Linux du composant SDK Platform-Tools`, validez les terms & Conditions

Dézzipez le dossier platform-tools-latest-linux.zip

Rendez vous dans le dossier platform-tools-latest-linux puis platform-tools.

Lancez les comande depuis ce dossier vous permettra d'avoir les derniere version d'ADB et de Fastboot.



Vérifiez la version d'ADB et de Fastboot avec les commandes citées plus haut.

\`\``bash

anthony@McFly-Bureau:~/platform-tools$ adb version

Android Debug Bridge version 1.0.41

Version 34.0.0-9570255

Installed as /usr/lib/android-sdk/platform-tools/adb





anthony@McFly-Bureau:~/platform-tools$ fastboot --version

fastboot version 34.0.5-10900879

Installed as /usr/lib/android-sdk/platform-tools/fastboot
