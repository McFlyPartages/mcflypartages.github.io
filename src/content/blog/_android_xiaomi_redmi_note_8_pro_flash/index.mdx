## Mettre a jour son Xiaomi Redmi Note 8 Pro (begonia)
Mettez a jour votre Redmi Note 8 Pro vers la versions v12.5 minimum (android 11) (la derniere version officielle de Xiaomi la 12.5.15.0.RGGEUXM, mais je n'ai pas réussie a passer au dessus de la v12.5.14.0)

Pour ma part je suis parti de la v12.0.4.0 QGGEUXM (Android 10), 
***Note :** C'est la roms Europe dite "Global" mais qui est replacé par la EEA depuis la version 12.0.6.0 aussi a destination de l'Europe.*

Alors deux choix s'offre a vous :
1. Jouer sur les régions pour récuperer de maniere oficiellle les dernieres roms (recommandé)
2. Récuperer les roms sur les lien ci dessous puis les installer depuis votre smartphone (elles seront verifiées avant d'etre installé.

### Depuis le téléphone
Pour mettre a jour directement depuis Xiaomi, il va falloir jouer avec les régions.

### Depuis les liens
***Note :** Cette méthode n'a pas marchée chez moi, les roms ne passaient pas le check. Mais je n'avais pas de carte SIM dans le telephone, peut etre est-ce lié a ca.*

Il va falloir récuperer la 12.0.6.0.QGGEUXM via ce lien https://xiaomifirmwareupdater.com/miui/begonia/stable%20beta/V12.0.6.0.QGGEUXM/ puis le coller dans votre téléphone.

Faite de meme avec la derniere version 12.5.15.0.RGGEUXM (2.4Go) recuperable ici https://xiaomifirmwareupdater.com/miui/begonia/stable/V12.5.15.0.RGGEUXM/


Ensuite, aller dans `Parametre -> A propos du téléphone` puis cliquer sur `Version MIUI` (carré image en haut a gauche)
Une fois affiché, appuyer 7 fois sur le logo, un popup doit vous afficher `des fonctionnalites de mise a jour supplemaentaires  sont disponnible`.

Cliquer maintenant sur les trois petits point en haut a droite puis `Selectionner une mise a jour a installer` selectionner le fichier telechargé precedemment (d'abord le v12.0.6.0) puis apres une verification il va l'installer et redemarrer.

## Via le recovery (TWRP) 

Démmarage sous recovery
Si looupe Bouton ON/OFF + Volume Haut

Vous pouvez autoriser TWRP a ne pas etre en lecture seule.

Selectionner Wipe puis format data taper `yes` puis valider.
> ATTENTION toutes vos données seront supprimées.

ensuite allez dans mount (montage) puis selectionner systeme et data; ensuite selectionner le stockage interne (ou SD) vous dreviriez avoir votre smarpthone reconnu sous linux (ou windaub)

Coller la roms custom de votre choix dans le dossier TWRP par exemple ou a la racine si vous preferez.


Une fois la rom coller, retourner sur la page principal du menu puis selectionnez `install` (installer) 

Ensuite selectionner votre roms.zip laisser les deux cases decochées.

switcher le slide d'en bas pour confirmer qur vous souhaitez flasher


A la fin vous pouvez faire wipe cache/Dalvik puis un reboot systeme.

Normalement vore telephone devrait booter sur votre custom rom fraichement installée.


## Via ADB
Entrer dans le recovery avec Bouton Onf/Off et volume haut
aller dans advanced puis sideload

cocher Wipe Davlvik Cache + Wipe Cache et sxipper le bouton du bas.
maintenant dans un Terminal, lancer la commande suivante (  adapter avec le nom de votre roms)
`adb sideload filename.zip`

## Revenir sur la rom MIUI
Pour revenir a la rom d'origine, il vous suffit soi de passer la TWRP et copier coller la derniere version de la roms Xiaomi pour votre smartphone ou alors via aADB.
